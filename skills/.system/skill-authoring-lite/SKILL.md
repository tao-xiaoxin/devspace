# Skill Authoring Lite

## What This Skill Does

Use this skill when creating or refactoring skills for DevSpace so they stay structured, concise, and reference-driven.

## Workflow

1. Keep `SKILL.md` focused on workflow and decisions.
2. Move detail-heavy material into `references/`.
3. Prefer examples and checklists over long prose.
4. Keep the response standard explicit.
